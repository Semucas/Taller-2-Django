# \# 🎓 Sistema de Gestión de Estudiantes - Django

# 

# Proyecto desarrollado para el \*\*Taller 2\*\* del curso de \*\*Plataformas de Programación Empresarial\*\*.

# 

# Sistema CRUD completo para gestionar estudiantes utilizando Django con arquitectura MVT (Model-View-Template).

# 

# \---

# 

# \## 📋 Descripción

# 

# Aplicación web que permite administrar información de estudiantes de manera eficiente. Implementa las cuatro operaciones CRUD (Create, Read, Update, Delete) con una interfaz moderna y responsiva utilizando Bootstrap 5.

# 

# \### \*\*Características principales:\*\*

# \- ✅ Arquitectura MVT (Model-View-Template)

# \- ✅ CRUD completo de estudiantes

# \- ✅ Interfaz responsiva con Bootstrap 5

# \- ✅ Panel de administración de Django

# \- ✅ Sistema de mensajes de éxito/error

# \- ✅ Validaciones de formularios

# \- ✅ Landing page informativa

# 

# \---

# 

# \## 🛠️ Tecnologías Utilizadas

# 

# \- \*\*Backend:\*\* Python 3.x, Django 5.1.4

# \- \*\*Frontend:\*\* HTML5, CSS3, Bootstrap 5.3, Bootstrap Icons

# \- \*\*Base de datos:\*\* SQLite3

# \- \*\*Control de versiones:\*\* Git \& GitHub
